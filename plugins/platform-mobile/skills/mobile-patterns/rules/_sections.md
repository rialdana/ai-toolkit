# Sections

This file defines all sections, their ordering, impact levels, and descriptions.
The section ID (in parentheses) is the filename prefix used to group rules.

---

## 1. Navigation (nav)

**Impact:** HIGH
**Description:** Mobile navigation patterns including stacks, tabs, and modals. Platform-agnostic principles that apply to all mobile frameworks.

## 2. Gestures (gesture)

**Impact:** MEDIUM
**Description:** Touch interaction patterns. Swipe actions, pull-to-refresh, and gesture conflicts.

## 3. Offline (offline)

**Impact:** MEDIUM
**Description:** Offline-first patterns. Caching, sync strategies, and graceful degradation.

## 4. Performance (perf)

**Impact:** MEDIUM
**Description:** Mobile performance patterns. List virtualization, image optimization, and memory management.
