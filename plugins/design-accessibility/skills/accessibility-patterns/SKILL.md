# Accessibility Patterns

Web accessibility best practices. Target WCAG 2.1 AA where practical. Many frameworks (shadcn, Radix) handle accessibility automatically.

## When This Applies

- Building any user interface
- Using custom components
- Reviewing for accessibility

## Quick Reference

| Section | Impact | Prefix |
|---------|--------|--------|
| Semantic HTML | HIGH | `semantic-` |
| Keyboard | HIGH | `keyboard-` |
| ARIA | MEDIUM | `aria-` |
| Visual | MEDIUM | `visual-` |

## Rules

See `rules/` directory for individual rules organized by section prefix.
