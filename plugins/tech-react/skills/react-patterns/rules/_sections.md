# Sections

This file defines all sections, their ordering, impact levels, and descriptions.
The section ID (in parentheses) is the filename prefix used to group rules.

---

## 1. Hooks (hooks)

**Impact:** HIGH
**Description:** React hooks are the foundation of modern React. Understand the rules of hooks and when to use each built-in hook.

## 2. Effects (effect)

**Impact:** HIGH
**Description:** useEffect is often misused. Most of the time, you don't need it. Understand when effects are appropriate.

## 3. Components (component)

**Impact:** MEDIUM
**Description:** React component patterns including function components, JSX patterns, and conditional rendering.

## 4. Performance (perf)

**Impact:** MEDIUM
**Description:** React-specific performance patterns including memoization, key props, and avoiding unnecessary re-renders.
